# Support

## How to Get Help

- Usage questions: open a GitHub Discussion or Issue
- Reproducible bugs: use the Bug Report template
- Security concerns: follow `SECURITY.md`

## What to Include

- SROS version (`sros --help` and `python -c "import sros; print(sros.__version__)"`)
- Python version and OS
- Minimal reproduction
- Relevant logs or receipt files

## Scope

Community support is best-effort for the hardened v1 surface.
Experimental or quarantined surfaces may change without compatibility guarantees.
